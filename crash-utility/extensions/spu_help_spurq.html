NAME
  spurq - shows contexts on the SPU runqueue

SYNOPSIS
  spurq  

DESCRIPTION
  This command shows info about all contexts waiting for execution 
in the SPU runqueue. No parameter is needed.

EXAMPLE
  Show SPU runqueue:

    crash> spurq
    PRIO[120]:
    c000000000fd7380
    c00000003bf31e00
    PRIO[125]:
    c000000039422d00
    c00000000181eb80
